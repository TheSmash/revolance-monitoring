Revolance | Jvm Monitoring

     Tracking several jvm memory usage has never been easier.

     launch: ./start.sh on host.yourcompany.com and start the jvms you want to track

        Notes:

            - Some executable jars are provided for you to test this product.
                To start them use: java -Xmx258 -jar ${jarFileName}.jar

     connect to: host.yourcompany.com:8080/jvm-monitoring-server with your favorite navigator

     Tadamm! You're now connected on the jvm monitoring server.

     Click on jvms (top right) and check which jvm you want to monitor and see the raw live memory statistics.
